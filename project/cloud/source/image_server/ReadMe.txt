本图片服务器的配置环境为：php5 + nginx1.4 + MySQL5.6 + GD2.0 + ImageMagick6.8 + imagick3.1 + PhpThumb
进行具体图片操作的是PhpThumb，
一个轻量级php图片操作工具，能够支持图片缩放、剪切、旋转和保存


get_image.php 中存放的是缩略图的实现代码
所有原始图片保存在 images/ 目录中，
所有缩略图保存在 images/thumbnails 中

首先判断指定文件名的指定规格(size)的文件是否存在（文件名为filename_$size.jpg），如果存在就直接显示并结束。
这一步没有判断原始文件是否存在，所以如果原始文件被删除，一定要删除它生成的各种大小的缩略图

如果指定大小的缩略图不存在，就对原始图片进行处理。如果生成的缩略图的尺寸小于原始图片，就保存之

get_image2.php 是另一个版本的实现，而且更加简洁
首先判断原始图片是否存在，如果不存在就显示错误信息并返回
判断指定大小的缩略图是否存在，如果存在就直接显示
如果缩略图不存在，就就对原始图片进行处理。如果生成的缩略图的尺寸小于原始图片，就保存之
